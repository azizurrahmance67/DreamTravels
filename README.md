#  Dream Travels - Bus Ticket Booking System

**Dream Travels** is a web-based bus ticket booking application built using **Python (Flask)** and **MS SQL Server**. This system allows passengers to select seats and receive a digital ticket through a demo payment gateway.

---

## Key Features

* **Seat Selection:** Users can interactively select their preferred seats.
* **Demo Payment Gateway:** Supports simulated payments via bKash, Nagad, and Cards.
* **E-Ticket Generation:** Automatically generates a professional digital ticket after a successful booking.
* **Admin Dashboard:** A dedicated dashboard for the admin to view all passenger details and booking history.
* **Database Management:** Securely stores booking data using MS SQL Server.

---

##  How to Run

1.  **Install dependencies:**
    ```bash
    pip install flask pyodbc
    ```
2.  **Setup Database:** Create a database named `DreamTravels` and setup the required tables (`Buses`, `Bookings`).
3.  **Run the application:**
    ```bash
    python app.py
    ```
4.  **Access the site:** Open `http://127.0.0.1:5000` in your browser.
